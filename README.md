A modern, responsive **Restaurant Website** designed for **Desi Vibes Cafe & Restaurant**, showcasing menu, events, gallery, and online table booking.

---

## 🌟 Project Overview

This project is a **fully responsive restaurant website** built using HTML, CSS, Bootstrap, and JavaScript.  
It provides an engaging UI to display restaurant services, menu, events, and contact details.

---

## 🚀 Features

### 🏠 Home Section
- Attractive hero carousel with animations
- Call-to-action buttons (Menu & Book Table)

---

### ℹ️ About Section
- Restaurant description
- Embedded video showcase
- Location details

---

### 🍴 Menu Section
- Categorized menu:
  - Drinks
  - Starters
  - Pasta
  - Burgers
  - Noodles
  - Tea
  - Desserts
- Dynamic filtering using **Isotope.js**

---

### 🎉 Events Section
- Event hosting details:
  - Birthday parties
  - Family gatherings
  - Engagement events
- Interactive slider using **Swiper.js**

---

### 📅 Table Booking
- User-friendly reservation form
- Fields:
  - Name, Email, Phone
  - Date & Time
  - Number of People
  - Message

---

### 🖼️ Gallery
- Responsive image grid
- Restaurant ambiance showcase

---

### ⭐ Testimonials
- Customer reviews with ratings
- Swiper-based slider UI

---

### 📍 Contact Section
- Google Maps integration
- Contact details:
  - Location
  - Email
  - Phone number
- Contact form

---

### 🔝 Additional Features
- Sticky navigation bar
- Smooth scrolling
- Back-to-top button
- Social media links
